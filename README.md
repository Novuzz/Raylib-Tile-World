# Raylib-Tile-World

## What is Raylib-Tile-World?

It's a Raylib project which create a tiled world, being able to walk in the tiles, place tiles and destroy tiles.
It have a Raycast collision system, so it prevents noclips, you may use it as reference

## How can I run it?

first you compile with this command
```
cmake --build build
```

And then you can find the .exe file in 
```
tileWorld\build\Debug
```
